# toronto-ems

Data from Open Data Toronto <https://www1.toronto.ca/wps/portal/contentonly?vgnextoid=1a66e03bb8d1e310VgnVCM10000071d60f89RCRD>

JSON conversion from Ogre <https://ogre.adc4gis.com/>
